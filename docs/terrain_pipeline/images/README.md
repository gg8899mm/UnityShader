# Images for QGIS Workflow Guide

This directory contains screenshots and diagrams referenced in the QGIS_Workflow.md guide.

## Required Images

1. **qgis_processing_toolbox.png** - Screenshot showing Processing Toolbox location in QGIS
2. **qgis_crs_settings.png** - Project Properties dialog with EPSG:3857 selection
3. **qgis_warp_settings.png** - GDAL Warp (reproject) dialog with key settings
4. **qgis_clip_settings.png** - Clip raster by extent dialog
5. **qgis_xyz_tiles_settings.png** - Generate XYZ tiles dialog with XYZ numbering option

## Adding Images

When creating these screenshots:
1. Use QGIS LTR with default theme
2. Highlight critical settings with red boxes or arrows
3. Ensure text is readable at normal viewing size
4. Include window chrome for context
5. Use English language interface
